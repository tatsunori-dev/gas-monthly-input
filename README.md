# 🌐 gas-monthly-input（GAS月次入力ツール）

> 🛠 **インストール不要・ブラウザだけで動く**軽貨物ドライバー向け月次管理ツールです。

Google Apps Script（GAS）で構築。  
ブラウザを開くだけで使えるため、**PCスペックや環境を選ばず**どこでも動作します。

---

## 😓 こんな課題を解決しました

- Pythonアプリはローカル環境が必要で、**別PCや外出先からアクセスできない**
- スプレッドシートでの管理は集計・レポート生成に手間がかかる
- 過去の売上CSV（移行前データ）を新ツールにシームレスに引き継げない
- 月・年単位の目標管理を一元化できていなかった

---

## 🚀 主な機能

- 🔐 ログイン認証（セッショントークン管理）
- 📝 日次入力・保存（Supabase へのリアルタイム保存）
- 🗂 GitHub 上の過去 CSV を自動読み込み・マージ
- 📈 月次レポ生成
  - 売上 / 稼働時間 / 時給 / TOP・WORST 分析
- 📅 年次レポ生成
  - 年間目標達成状況・月別サマリ
- 🎯 月ごとの個別目標管理
- 💾 CSV インポート / データ削除

---

## 🛠 技術構成

| 項目 | 内容 |
|------|------|
| バックエンド | Google Apps Script（GAS）|
| フロントエンド | HTML / CSS / JavaScript |
| DB | Supabase（PostgreSQL）REST API |
| データ連携 | GitHub raw CSV 自動読み込み・マージ |
| 認証 | セッショントークン管理 |

---

## 🎯 一言アピール

「Pythonアプリが使えない環境でも同じデータを見たい」という要求から生まれたツール。  
GAS × Supabase × GitHub CSVという構成で、**インストール・サーバー費用ゼロ**を実現。  
**複数ツールを横断した一貫したデータ管理体制**を構築しました。
